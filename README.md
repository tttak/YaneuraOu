# About this project

NNUE評価関数に空きマス位置の特徴量を追加したものです。

## 強さ

手元の学習ではオリジナルのTNKよりかなり弱い

- 学習の方法が分かっていない
  - 私自身オリジナルのNNUE評価関数の0からの学習に成功していない
- 特徴量の追加による計算コスト増が表現力の増加によるプラスを上回っている可能性もある
